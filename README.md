JavaScript Number Guessing Game
Project Overview-
This is a simple console-based number guessing game built using JavaScript. The program generates a random number within a user-defined range, and the user attempts to guess the correct number. Hints are provided after each attempt to guide the user.

How It Works-
1.The user enters a maximum number.
2.The program generates a random number between 1 and the given maximum.
3.The user keeps guessing the number.
4.If the guess is too high or too low, a hint is displayed.
5.The game continues until the correct number is guessed or the user types "quit" to exit.

Technologies Used-
-JavaScript
-Math.random() for random number generation
-Loops and conditional statements
-User input using prompt()
